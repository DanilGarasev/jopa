Исходный код проекта группы 15.ИСиП.21.ОФ.С.1. Сайт HAVOK стоит на WordPress, полностью редактируется при помощи CarbonFields. Заявки с формы выходят на электронную почту администратора сайта. При вёрстке были использованы технологии: html,css3. Back-end полностью на PHP с подключенной по mySQL базой данных. Посмотреть сайт в интернете можно по ссылке: http://f0748076.xsph.ru/
Админ панель доступна по ссылке: http://f0748076.xsph.ru/wp-admin/
